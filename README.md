## My To-Do List Web Application
A simple and responsive To-Do List application built to help users manage daily tasks efficiently.

## Features
- Add new tasks
- Delete completed tasks
- Clean and user-friendly interface
- Responsive design 

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose 
This project was built to practice front-end web development fundamentals, including DOM manipulation, and basic UI design
